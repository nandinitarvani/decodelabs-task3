# 🔐 Project 3 – Random Password Generator

This project was developed as part of my internship at DecodeLabs.

# 📌 Project Overview

A Python-based Random Password Generator that creates strong and secure passwords based on user-defined length and character preferences. The application generates random passwords using letters, numbers, and optional special characters.

# 🚀 Features

* Generate random passwords of custom length
* Option to include or exclude special characters
* Input validation using exception handling
* Password length validation
* Uses randomization for secure password generation
* User-friendly console interface

# 🛠 Technologies Used

* Python
* Random Module
* String Module
* Loops
* Conditional Statements
* Exception Handling

# 📚 Learning Outcomes

This project strengthened my understanding of Python modules, string manipulation, randomization techniques, input validation, exception handling, and building practical security-focused applications.
